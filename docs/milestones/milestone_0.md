# M0 Definition of Done - Player Controller Test Harness

M0 is complete when:

- Player scene is reusable in other levels.
- Player can move left/right on a locked 2.5D plane.
- Player can jump with coyote time and jump buffering.
- Movement values are exposed for tuning.
- Camera follows smoothly using an orthographic side-view setup.
- Test level includes flat ground, gaps, slopes, platforms, and collision edge cases.
- Player can traverse the test level without obvious collision bugs.
- Debug overlay can show movement state and grounded status.
- Controller can be dropped into M1 forest slice without rewriting it.
