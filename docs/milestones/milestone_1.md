# Milestone 1 - Forest Slice

## Goal
Prove that the chosen low-poly layered forest style is achievable, readable, and fun to build.

## Deliverable
A 30-second playable side-scroller scene with player movement, one enemy, modular terrain, layered background, and basic environmental dressing.

## Required Features
- Orthographic 2.5D camera
- Player controller placeholder or existing controller
- Modular terrain kit
- Forest background layers
- Scatter prop kit
- One enemy
- One collectible or interactable
- Basic lighting/fog/atmosphere
- Screenshot pass

## Success Criteria
- Scene looks appealing in a still screenshot.
- Player is readable at all times.
- Enemy is readable at all times.
- Terrain can be extended using reusable pieces.
- At least 80% of visible art is made from reusable assets.
- No major visual reliance on neon/glow.
